# SadServers 解法集

https://sadservers.com/

## SadServersとは

- Linux技術者向けの問題集。問題のあるサーバが作られるのでサーバ上で修正して問題を解く
- 主にUNIXコマンドやOS, ミドルウェア周りの知識が問われる
- サーバはAWS上に作成され、SadServers側にコストがかかるのである程度時間が経過するとshutdownされる
- 1日に何問か解いていると24時間後まで解答できなくなったりする
